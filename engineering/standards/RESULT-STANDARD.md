# RESULT STANDARD

## Purpose

The calculator result is the most important part of the page.

Users should never need to search for the answer.

---

## Requirements

Results should contain:

✓ Primary Result

✓ Secondary Results (when applicable)

✓ Copy Button

✓ Clear Labels

✓ Proper Units

---

## Presentation

The result should be visually dominant.

The result should receive more visual emphasis than the calculator inputs.

---

## Optional Enhancements

Where appropriate include:

- Explanation
- Practical meaning
- Related calculation
- Next recommended tool

---

## Avoid

Tiny text

Hidden results

Multiple competing result boxes

Ambiguous units
