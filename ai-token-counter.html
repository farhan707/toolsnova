What are tokens and why do they matter?

🔤 What is a token?
Tokens are the fundamental units LLMs process. Common short words = 1 token. Longer or rare words = 2–4 tokens. Numbers and punctuation each count. Rule of thumb: 1 token ≈ 0.75 words ≈ 4 characters in English.
💸 Input vs Output cost
Every model charges separately for input (your prompt) and output (the response). Output is always 3–5× more expensive than input. GPT-4o: $2.50 input vs $10.00 output per MTok — a 4× difference.
🔄 Prompt Caching
Claude and GPT-4o cache repeated system prompts at up to 90% discount. Claude cached reads: $0.30/MTok vs $3.00/MTok uncached. For apps with fixed system prompts, caching cuts input costs by 50–90%.
📐 Context Window
The context window is the total tokens a model can process at once (input + output combined). GPT-4o: 128K. Claude 3.5: 200K. Gemini 2.0 Pro: 1M tokens. Exceeding it truncates older parts of the conversation.
✂️ Reduce token usage
Remove filler words from system prompts. Use bullet points instead of prose instructions. Summarize chat history instead of sending everything. Use a smaller model for simple tasks — GPT-4o Mini is 15× cheaper than GPT-4o.
🏆 Cheapest 2026 models
Gemini 2.0 Flash: $0.10/MTok input. GPT-4o Mini: $0.15/MTok. DeepSeek V3: $0.27/MTok. For most tasks these match GPT-4o quality at 10–25× lower cost. Use this counter to find your cheapest option.
💡 Token math: 1,000 English words ≈ 1,333 tokens. A typical system prompt (300 words) ≈ 400 tokens. A full A4 page of text ≈ 500 tokens. A book chapter (5,000 words) ≈ 6,650 tokens. At GPT-4o pricing ($2.50/MTok), processing 1,000 tokens costs just $0.0025 — but at scale (1M calls/month × 1,000 tokens) that's $2,500/month.
