📋 Prompt
You are a senior SEO strategist who builds keyword strategies that dominate niche topics.

Seed keyword: [seed keyword]
Niche: [website niche]
Audience: [target audience]
Goal: [content goal — traffic/leads/sales/authority]

Task: Build a complete keyword research report:

1. SEED KEYWORD ANALYSIS: Search intent, competition level, best page type

2. KEYWORD CLUSTERS (8–10 clusters): For each cluster:
- Cluster theme
- Primary keyword (pillar page)
- 5–8 supporting keywords (sub-pages)
- Search intent for each
- Suggested page title

3. CONTENT ARCHITECTURE:
- Hub page (pillar)
- Spoke pages (clusters)
- Internal linking structure

4. QUICK WIN KEYWORDS: 10 long-tail, high-intent, low-competition keywords with explanation of why each is a quick win

5. FEATURED SNIPPET OPPORTUNITIES: 5 question-format keywords + content format to target each

Format: Tables where possible. Priority order for each cluster (High/Medium/Low).
📄 Example output
SEED KEYWORD: 'AI tools for business'
Search Intent: Informational/Commercial
Best Page Type: Comparison listicle + hub page

CLUSTER 1: Free AI tools (Priority: HIGH)
Pillar: 'Best free AI tools for business 2026'
Supporting keywords:
- 'free AI writing tools' (informational, high vol)
- 'free AI image generators' (informational)
- 'ChatGPT alternatives free' (commercial)
- 'free AI tools for small business' (commercial)
- 'no-cost AI productivity tools' (informational)

CLUSTER 2: AI tools by function (Priority: HIGH)
Pillar: 'AI tools for marketing 2026'
Supporting keywords:
- 'AI copywriting tools'
- 'AI social media tools'
- 'AI email marketing tools'...

💡 Pro Tips
Don't chase volume alone — a 200 search/month keyword with low competition beats a 10,000/month keyword you can't rank for
Cluster your keywords by topic, not just by word similarity — semantic grouping is what builds topical authority
Target featured snippets first — they give you position 0 with a fraction of the effort of ranking #1
Use your clusters to build a publishing calendar — one cluster per month for 12 months is a solid SEO strategy
⚠️ Common Mistakes
Targeting head terms (single words) as a new site — you'll never rank for 'SEO' but can rank for 'SEO for e-commerce beginners'
Ignoring search intent — ranking for the wrong intent means bounce rates that kill your rankings
Not updating keyword research — the competitive landscape shifts every 6–12 months
Choosing keywords without checking the actual SERP — sometimes the top results tell you the true intent
❓ FAQ
  • Do I still need a paid keyword tool?
    Paid tools (Ahrefs, Semrush) give you accurate volume and difficulty data that AI cannot provide. Use this prompt to build your strategy and cluster structure, then validate volumes in a real keyword tool.
  • How many keywords should I target per page?
    One primary keyword and 5–10 semantically related keywords per page. Don't try to optimise one page for unrelated terms.
  • How long does it take to rank?
    For a new domain, expect 6–12 months for competitive terms. Quick-win long-tail keywords can rank in 4–8 weeks. Topical authority compounds over time — the longer you publish, the faster each new page ranks.
  • Can Gemini help with keyword research?
    Yes. Gemini 2.0 Flash has strong web awareness and can suggest keyword ideas based on current trends. For actual search volume data, verify in Google Search Console or a dedicated tool.
