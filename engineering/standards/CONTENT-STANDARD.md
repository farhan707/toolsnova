# Content Standard

## Tone

Professional.

Simple.

Educational.

Accurate.

---

## Writing Rules

Use short paragraphs.

Use headings.

Use tables whenever appropriate.

Use bullet lists.

Avoid repetition.

Avoid marketing language.

Avoid keyword stuffing.

---

## AI Rules

Every section should answer one question.

Every definition should be self-contained.

Quick Answer should summarize the page.

Examples should use realistic values.

FAQs should answer genuine user intent.
