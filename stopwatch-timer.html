Frequently Asked Questions

What is the difference between a stopwatch and a countdown timer?
A stopwatch counts up from zero, measuring elapsed time. A countdown timer counts down from a set time to zero. Both are useful for different tasks — stopwatches for timing events, laps, or tasks; countdown timers for setting time limits, cooking, studying, or workouts.
Does the timer work in the background?
Yes. The timer continues running even when you switch to another tab. The page title does not update for the stopwatch (to avoid performance issues), but the countdown timer audio alert fires even if the tab is in the background.
What is the lap feature?
The Lap button records the current stopwatch time as a split time without stopping the clock. This is useful for timing multiple intervals — like laps in a race or rounds in a workout — while keeping the total elapsed time running.
