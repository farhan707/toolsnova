About lot size calculation

Risk management
Never risk more than 1–2% per trade. This calculator ensures your position size matches your risk tolerance.
Pip value
For standard EUR/USD lots, pip value is ~$10. For gold (XAUUSD), use $1 per 0.01 move = $100/lot for $1 move.
Formula
Lot size = Risk Amount / (Stop Loss in pips × Pip Value per lot)
