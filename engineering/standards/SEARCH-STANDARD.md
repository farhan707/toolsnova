# Search Intelligence Standard

Optimize for topics, not keywords.

---

## Metadata

Unique title.

Unique meta description.

Canonical URL.

Open Graph.

Twitter Card.

---

## Schema

Use only valid schema.

Prefer:

- WebPage
- SoftwareApplication
- FAQPage
- BreadcrumbList

---

## Internal Linking

Link naturally.

Never force links.

---

## AI Readiness

Use clear headings.

Use explicit definitions.

Use examples.

Use comparison tables.

Use entity relationships.
