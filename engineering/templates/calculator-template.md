# Calculator Template

## Purpose

This template defines the standard structure for all calculator pages on ToolsNova.

Examples:

- Margin Calculator
- Profit Calculator
- ROI Calculator
- Loan Calculator
- EMI Calculator
- Discount Calculator
- Percentage Calculator

---

# Standard Page Structure

1. Hero

2. Calculator

3. Quick Result

4. Quick Answer

5. Definition

6. Formula

7. Worked Examples

8. Practical Use Cases

9. Common Mistakes

10. FAQ

11. Related Tools

12. References

13. Last Updated

---

# Calculator Requirements

Every calculator should:

- Calculate instantly
- Validate user input
- Support keyboard navigation
- Display clear error messages
- Show properly formatted results
- Display units when applicable
- Allow easy copying of results

---

# Educational Content

Every calculator should explain:

- What the calculation means
- When to use it
- How the formula works
- Real-world examples
- Common mistakes

---

# AI Readiness

Every calculator should contain:

- Clear definitions
- Formula explanation
- Worked examples
- FAQs
- Semantic headings
- Entity-rich content

---

# Accessibility

Use semantic HTML.

Associate every label with an input.

Support keyboard navigation.

Maintain proper heading hierarchy.

---

# Definition of Success

The calculator should answer both:

"I need an answer."

and

"I want to understand the answer."
