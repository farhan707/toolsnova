# HERO STANDARD

## Purpose

The hero is the first thing every visitor sees.

Within five seconds the user should understand:

- What this tool does.
- Why they should trust it.
- Why ToolsNova is a better choice.

---

## Required Elements

Every Tier-1 page hero should contain:

✓ H1

✓ One sentence description

✓ 3–5 key benefits

✓ Primary CTA (Calculator)

✓ Optional secondary CTA (Copy, Share, etc.)

---

## Hero Benefits

Benefits should be short.

Examples:

- Instant results
- Accurate formulas
- Works offline
- No signup required
- Mobile friendly
- Browser-based
- Free forever

---

## Rules

Do not use marketing hype.

Do not repeat keywords.

Communicate value immediately.
