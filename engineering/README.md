# ToolsNova Engineering System

This directory contains the engineering standards used for building every page on ToolsNova.

## Structure

MASTER_ENGINEERING_PROMPT.md
Global AI engineering workflow.

standards/
Engineering rules.

pages/
Page specifications.

templates/
Reusable page blueprints.

## Workflow

1. Read MASTER_ENGINEERING_PROMPT.md

2. Read all standards.

3. Read page specification.

4. Analyze target file.

5. Implement.

6. QA.

7. Return GitHub Pull Request.
